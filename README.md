# 🧑‍💻 My Portfolio

> 🇬🇧 English version

Personal portfolio built with Next.js 14, TypeScript, and Tailwind CSS. Includes three sections, a bilingual interface (English / Spanish), dark and light mode, and a customizable accent color.

The portfolio includes three pages:

- 🏠 **Home** — Introduction, background, academic info, soft skills and degrees
- 💼 **Work** — Project showcase with descriptions and category filters (Desktop / Mobile)
- ⚙️ **Tech** — Technologies organized by category: Languages & Tools, Frontend, and Backend

Notable features:

- 🌐 **Bilingual** — Full English and Spanish support, persisted in localStorage
- 🌗 **Dark / Light mode** — Toggleable theme persisted in localStorage
- 🎨 **Custom accent color** — Choose between Emerald, Rose, Blue, and Yellow
- 📱 **Responsive** — Desktop and mobile versions for Home and Work pages

> ⚠️ Tech page is desktop only.

### Stack

- ⚡ **Next.js 14** — App Router & framework
- 🔷 **TypeScript** — Type safety
- 🎨 **Tailwind CSS** — Styling
- 🧩 **MUI (Material UI)** — Components
- 🪄 **React Icons** — Icon set

# 🧑‍💻 Mi Portfolio

> 🇦🇷 Versión en español

Portfolio personal construido con Next.js 14, TypeScript y Tailwind CSS. Incluye tres secciones, una interfaz bilingüe (inglés / español), modo oscuro y claro, y un color de acento personalizable.

El portfolio incluye tres páginas:

- 🏠 **Home** — Introducción, trayectoria, información académica, habilidades blandas y títulos
- 💼 **Work** — Showcase de proyectos con descripciones y filtros por categoría (Escritorio / Móvil)
- ⚙️ **Tech** — Tecnologías organizadas por categoría: Lenguajes & Herramientas, Frontend y Backend

Características destacadas:

- 🌐 **Bilingüe** — Soporte completo en inglés y español, persistido en localStorage
- 🌗 **Modo oscuro / claro** — Tema toggleable persistido en localStorage
- 🎨 **Color de acento personalizable** — Elegí entre Esmeralda, Rosa, Azul y Amarillo
- 📱 **Responsive** — Versiones desktop y mobile para las páginas Home y Work

> ⚠️ La página Tech es solo para desktop.

### Stack

- ⚡ **Next.js 14** — App Router y framework
- 🔷 **TypeScript** — Tipado estático
- 🎨 **Tailwind CSS** — Estilos
- 🧩 **MUI (Material UI)** — Componentes
- 🪄 **React Icons** — Set de íconos
